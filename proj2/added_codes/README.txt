Foi necessário utilizar as funções "copy_to_user" e "copy_from_user" na syscall para manipular endereços que foram fornecidos pelo usuário. Dessa forma, é possível verificar se o endereço é, de fato, do usuário (e não é, por exemplo, do kernel); além disso, caso o endereço seja inacessível, um erro é retornado e o endereço não é acessado (o que previne um kernel panic, caso o endereço seja inválido).
